Software

Use our available software to enhance your data publishing and querying

Substring fragmenter

This program creates TREE fragmentations from RDF datasets. These fragmentations can be queried over using the TreeMunica query engine.

Documentation on Github

TREEmunica

This is a fork of the Comunica Knowledge Graph querying library to support TREE and LDES metadata extraction, and perform for example autocompletion across a tree:Collection that is fragmented using substrings.

Documentation on Github

The Linked Data Event Streams Client

This library uses Comunica components to create a NodeJS tool for replicating a full Linked Data Event Stream and stay in sync with it.

Documentation on Github
